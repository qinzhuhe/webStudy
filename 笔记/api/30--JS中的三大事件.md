# 鼠标事件

- click--->单击

- dblclick--->双击

- mousedown--->鼠标按下

- mouseup--->鼠标抬起

- mouseover--->鼠标进入

- mouseout--->鼠标离开

  (这种方法，在鼠标进入或离开作用元素或其子元素时，都会触发)

- mouseenter--->鼠标进入

- mouseleave--->鼠标离开

  (这种方式，只有当鼠标进入作用元素时才会被触发)


# 键盘事件

- keydown--->键盘按下
- keyup--->键盘抬起
- keypress--->键盘按下抬起/键盘使用

# HTML事件

- load--->文档加载完成
- select--->被选中的时候
- change--->内容被改变
- focus--->得到光标
- resize--->窗口尺寸变化
- scrool--->滚动条移动
