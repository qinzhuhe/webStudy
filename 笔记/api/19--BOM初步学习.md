`Javascript`分三个部分：

1.`ECMAScript`标准---基本语法

2.`DOM`---`Document Object Model`  文档对象

3.`BOM`---`Borwser Object Model`  浏览器对象



浏览器中有个顶级对象：`window`

页面中顶级对象：`document`

页面中所有的内容都是属于浏览器的，页面中的内容都是属于`window`的



变量是`window`的，函数也是`window`的
