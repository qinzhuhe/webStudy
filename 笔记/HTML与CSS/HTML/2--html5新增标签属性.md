

header`-->定义页面头部，页眉

`nav`-->定义导航栏

`footer`-->定义页面底部，页脚

`article`-->定义文章内容

`selection`-->定义区域

`aside`-->定义其所处内容之外的内容，侧边

`datalist`-->标签定义选项列表，与`input`配合使用

```html
学习科目：<input type="text" placeholder="请选择学院" list="xy">
			<datalist id="xy">
				<option>HTML</option>
				<option>CSS</option>
				<option>JavaScript</option>
			</datalist>
```

`fieldest`-->可将表单内的相关元素分组、打包，与`legend`搭配使用

```html
<fieldset>
			<legend>学生档案</legend>
</fieldset>
```
