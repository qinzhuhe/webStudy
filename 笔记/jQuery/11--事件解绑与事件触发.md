## 事件解绑：`off()`

1. ```javascript
   //解绑匹配元素的所有事件
   $(selector).off();
   ```

2. ```javascript
   //解绑匹配元素的所有click事件
   $(selector).off("click")
   ```

## 触发事件： 

1. ```javascript
   $(selector).click();//触发 click 事件
   ```

2. ```javascript
   $(selector).trigger("click")//触发click事件
   ```
