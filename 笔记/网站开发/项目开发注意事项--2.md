1. **首先在此总结一下目前常见的网页开发技术（硬性）**

	- HTML -- 网页内容结构（GUI）
	- CSS -- 网页外观样式（GUI）
	- JavaScript -- 编程语言（可以用于调用浏览器提供的API）
	- Web APIs -- 网页交互（界面功能）
	- jQuery等 -- 便携手段 （不算必要）

2. **编程能力/编程思想/解决问题的思路（软性）**
- 我要做什么（我要得到什么），我目前有什么（我能拿到什么）。

至此便能独立的完成一个网页的开发，而具体能完成的东西就是一个个的网页，而且能给页面中添加一些交互功能，但是距离完成一个成熟的网站还有这一段距离。
